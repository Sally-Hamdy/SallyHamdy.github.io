# How to publish the new portfolio

## Preview first
Double-click `index.html` in this folder — it opens in your browser with all images working.

## Prepare the 3 logo files
Save the logo images with these EXACT names (lowercase):
- `dupack.png` — the Dupack / Dubai Packaging Industry logo
- `empack.png` — the Empack logo
- `linsync.png` — the LinSync Technologies logo

## Upload to GitHub (4 files)
All other images already exist in the repo. Upload `index.html` + the 3 logo files:

1. Go to **github.com** and sign in (account: **Sally-Hamdy**, likely with salsabeel.h.mohamed60@gmail.com — use "Forgot password" if needed).
2. Open the repo: **github.com/Sally-Hamdy/SallyHamdy.github.io**
3. Click **Add file → Upload files**.
4. Drag `index.html`, `dupack.png`, `empack.png`, and `linsync.png` into the page.
5. Click **Commit changes** (it will replace the old index.html).
6. Wait 1–2 minutes, then open https://sally-hamdy.github.io/SallyHamdy.github.io/ and press **Ctrl+F5** to see the new design.

The old `style.css` and `script.js` are no longer used (everything is inside the new index.html) — you can leave them or delete them.
